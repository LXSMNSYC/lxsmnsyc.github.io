    MaybeEmitter

    Extends:

    rx-cancellable~Cancellable → MaybeEmitter

    Abstraction over a MaybeObserver that allows associating a resource with it.

    Calling onSuccess(Object) multiple times has no effect. Calling onComplete() multiple times has no effect. Calling onError(Error) multiple times has no effect.
