    public add(cancellable: Cancellable): boolean source

    Adds a Cancellable as a child Cancellable.

    Child Disposables are cancelled, enabled and disabled if their parent Cancellable is cancelled as well.

    A child Cancellable cannot have more than one parent Cancellable.

    If a child Cancellable is added to a parent Cancellable that is already cancelled, the child Cancellable is cancelled.
