ProcessorCallback(…inputs) → {*}

Source:
  • index.js, line 39

A callback processor which receives the input from input Relays, process the inputs, and passes the output to output Relays.

If the processor is an async function or a function returns a Promise-like object, the result is passed on fulfillment of the object.
