    public add(disposable: Disposable): boolean source

    Adds a Disposable as a child Disposable.

    Child Disposables are disposed, enabled and disabled if their parent Disposable is disposed as well.

    A child Disposable cannot have more than one parent Disposable.

    If a child Disposable is added to a parent Disposable that is already disposed, the child Disposable is disposed.
