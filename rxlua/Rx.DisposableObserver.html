DisposableObserver

Rx. DisposableObserver

An abstract Rx.Observer that allows asynchronous cancellation by implementing Disposable.

Use the public dispose() method to dispose the sequence from within an onNext implementation.

Like all other consumers, DisposableObserver can be subscribed only once. Any subsequent attempt to subscribe it to a new source will yield a ProtocolViolation with messages:

  • "It is not allowed to subscribe with a(n) multiple times."
  • "Disposable already set!.

Implementation of onStart(), onNext(Object), onError(Throwable) and onComplete() are not allowed to throw any unchecked exceptions.
