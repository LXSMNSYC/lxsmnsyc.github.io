retry(predicateopt) → {Promise}

Source:
  • px.js, line 154

Retries a rejected DeferredPromise multiple times until a resolved value is fulfilled.

If a function is provided, Retries until the function returns false, in which will return a rejected Promise with the propagated error.

Example
Promise.reject(50).defer().retry();
